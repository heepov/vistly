# Vistly Bot

Telegram-бот для поиска, отслеживания и организации фильмов и сериалов.

## Технологии
- Python
- PostgreSQL (Peewee ORM)
- aiogram
- aiohttp
- environs

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` и заполните переменные окружения.
3. Запустите бота.
